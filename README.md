# E-Commerce Country Dashboard

> **Note:** This project was originally created during my mid-year break, around early August, when I first started learning Power Pivot and Excel data modeling.

This repository contains an interactive Excel-based **E-Commerce Country Dashboard** built using **Power Pivot**, **Power Query**, and **Pivot Tables**. The project visualizes e-commerce performance based on selected product categories and countries.

The goal of this project is to transform raw e-commerce data into meaningful insights for business decision-making.
